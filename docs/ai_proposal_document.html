(요약)		 머신러닝과 자연어처리 기술을 활용하여 의료인 국가시험 문제의 자동 생성, 난이도 조정, 품질 검증을 지원하는 지능형 시스템을 구축하여 출제위원의 업무 효율성을 50% 향상시키고 문제 품질의 일관성을 확보하는 혁신적 솔루션

□ 현황 및 문제점

○ 출제 업무의 현황

현재 국시원은 연간 의사, 치과의사, 한의사, 약사, 간호사 등 총 5개 직종의 국가시험을 실시하고 있으며, 각 시험마다 평균 300-400문항의 문제를 출제해야 합니다.

○ 주요 문제점

  • 높은 업무 부담: 출제위원들이 문제 작성, 검토, 수정에 과도한 시간 소요 (위원 1인당 월 평균 40시간)
  • 품질 편차: 출제위원 개인의 역량에 따른 문제 품질 및 난이도 편차 발생
  • 중복 문제: 과거 출제 문제와의 중복성 검토에 많은 시간 소요
  • 표준화 부족: 문제 유형, 난이도, 출제 형식의 일관성 부족
  • 신규 위원 교육: 출제 경험이 부족한 신규 위원의 적응 기간 장기화
핵심 문제: 수작업 중심의 출제 프로세스로 인한 비효율성과 품질 불일치 문제가 지속적으로 발생하고 있으며, 출제위원 확보와 관리에 어려움이 증가하고 있습니다.
□ 개선방안

○ AI 기반 출제 지원 시스템 구축

1단계: 문제 생성 AI 모듈

  • 학습 데이터 구축: 과거 10년간 출제된 모든 문제 데이터를 학습
  • 자연어처리 기술: GPT 기반 언어모델을 의료 전문 용어로 특화 훈련
  • 문제 자동 생성: 출제 기준과 난이도를 입력하면 초안 문제 자동 생성
  • 보기 생성 AI: 정답과 오답 보기를 논리적으로 구성하는 알고리즘

2단계: 난이도 조정 시스템

  • IRT 모델 적용: 문항반응이론을 활용한 문제 난이도 자동 측정
  • 예측 모델: 과거 데이터 기반으로 문제별 정답률 예측
  • 균형 조정: 전체 시험의 난이도 분포를 자동으로 균형 조정

3단계: 품질 검증 모듈

  • 중복성 검사: 벡터 유사도 분석으로 기존 문제와의 중복 자동 탐지
  • 오류 검증: 문법, 의학적 정확성, 논리적 오류 자동 검증
  • 가이드라인 준수: 출제 기준 및 형식 자동 검토

○ 시스템 구현 방안

기술 스택:

  • AI 엔진: OpenAI GPT-4 API + 의료 전문 Fine-tuning
  • 백엔드: Python Django + PostgreSQL
  • 프론트엔드: React.js + TypeScript
  • 클라우드: AWS EC2 + RDS + S3
  • 보안: OAuth 2.0 + 데이터 암호화

○ 단계별 도입 계획

  • 1단계 (3개월): 시스템 설계 및 기본 AI 모델 개발
  • 2단계 (3개월): 파일럿 테스트 (간호사 시험 대상)
  • 3단계 (6개월): 전 직종 확대 적용 및 시스템 고도화

○ 정성적 효과

  • 업무 효율성 향상: 출제위원이 창의적 문제 개발에 더 집중 가능
  • 품질 표준화: 모든 문제의 품질과 난이도 일관성 확보
  • 신속한 대응: 긴급 시험 일정 변경 시 빠른 문제 준비 가능
  • 데이터 축적: 문제 출제 패턴 분석을 통한 지속적 개선
  • 위원 만족도: 반복적 업무 감소로 출제위원 참여 의욕 증진

○ 사회적 효과

  • 시험 신뢰도 제고: 일관된 품질의 문제로 공정성 향상
  • 수험생 만족도: 예측 가능한 난이도와 명확한 문제로 시험 준비 효율성 증대
  • 의료계 발전: 더 나은 의료인 선발을 통한 의료 서비스 질 향상
핵심 성과: 연간 약 8,000시간의 출제위원 업무시간 절약과 3억원의 직접적 비용 절감 효과가 예상되며, 문제 품질의 표준화로 시험 신뢰도가 크게 향상될 것입니다.
□ 장애요인 및 극복방안

○ 예상 장애요인

1. 기술적 장애요인

  • 의료 전문성: AI가 의학적 정확성을 완벽히 보장하기 어려움
  • 한국어 처리: 의료 전문 용어의 정확한 자연어처리 한계
  • 창의성 부족: AI 생성 문제의 창의성과 참신성 부족

2. 조직적 장애요인

  • 출제위원 저항: 기존 출제위원들의 AI 도입 거부감
  • 예산 확보: 초기 시스템 구축 비용 부담
  • 법적 검토: AI 생성 문제의 법적 책임 소재 불분명

○ 극복방안

1. 기술적 극복방안

  • 전문가 협업: AI는 초안 생성만 담당하고 최종 검토는 출제위원이 수행
  • 점진적 학습: 의료진 피드백을 통한 지속적 AI 모델 개선
  • 하이브리드 방식: AI 지원 + 인간 전문성 결합 모델 채택
  • 품질 보증: 3단계 검증 프로세스 (AI → 출제위원 → 외부검토)

2. 조직적 극복방안

  • 단계적 도입: 파일럿 테스트 → 부분 도입 → 전면 확산
  • 교육 프로그램: 출제위원 대상 AI 활용 교육 실시
  • 인센티브 제공: AI 활용 우수 위원 포상 제도
  • 투자 회수: 3년 내 비용 절감으로 투자 비용 회수 가능
  • 법적 대비: AI 지원 도구로 명시하여 최종 책임은 출제위원이 담당

○ 위험 관리 계획

  • 백업 시스템: 기존 출제 방식 병행 운영으로 안정성 확보
  • 보안 강화: 문제 유출 방지를 위한 강화된 보안 체계
  • 지속적 모니터링: 시스템 성능과 문제 품질 실시간 모니터링
성공 전략: AI를 완전 대체 도구가 아닌 '지능형 보조 도구'로 포지셔닝하여 출제위원의 전문성을 보완하는 방향으로 접근함으로써 저항을 최소화하고 실용성을 극대화할 계획입니다.
